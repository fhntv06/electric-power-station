# Методы

## Общие методы
### Start()
<blockquote>
  <p>Метод запускающийся при старте игры</p>
</blockquote>

### Update()
<blockquote>
  <p>Метод запускающийся количество раз равное количеству кадров игры например, при просадке производительности будет 40 запусков в секунду, а при нормальной работе 60.</p>
</blockquote>

### FixedUpdate()
<blockquote>
  <p>Метод запускающися фиксированное количество раз количество раз настраивается в настройках Unity</p>
</blockquote>

## Методы для игровых объектов
  ### SetActive()
  <blockquote>
    <p>Метод управляет активностью объекта</p>
    <h3>Параметры:</h3>
    <ul>
      <li><p>true - показывает объект на сцене</p></li>
      <li><p>false - скрывает объект со сцены</p></li>
    </ul>
  </blockquote>
